## 파이썬이란

파이썬(Python)은 1991년 귀도 반 로섬(Guido van Rossum)이 개발한 범용 프로그래밍 언어입니다. 문법이 간결하고 읽기 쉬워 프로그래밍을 처음 배우는 사람에게 가장 많이 추천되는 언어 중 하나입니다. 현재 웹 개발, 데이터 분석, 인공지능, 자동화 스크립트 등 다양한 분야에서 널리 사용되고 있으며, 전 세계적으로 가장 인기 있는 프로그래밍 언어 중 하나로 자리 잡았습니다.

파이썬의 가장 큰 특징은 **들여쓰기(Indentation)** 를 통해 코드 블록을 구분한다는 점입니다. 중괄호 대신 들여쓰기를 사용하기 때문에 코드가 깔끔하고 가독성이 뛰어납니다.

## 파이썬을 배워야 하는 이유

파이썬을 배워야 하는 이유는 여러 가지가 있습니다.

- **낮은 진입 장벽**: 영어 문장을 읽는 것처럼 자연스러운 문법 구조를 가지고 있어 초보자도 쉽게 접근할 수 있습니다.
- **풍부한 라이브러리**: NumPy, Pandas, Django, Flask, TensorFlow 등 방대한 라이브러리 생태계를 보유하고 있습니다.
- **높은 수요**: 국내외 IT 기업에서 파이썬 개발자에 대한 수요가 꾸준히 증가하고 있습니다.
- **다양한 활용 분야**: 웹 개발부터 데이터 과학, 머신러닝, 업무 자동화까지 폭넓게 활용됩니다.
- **활발한 커뮤니티**: 전 세계적으로 거대한 개발자 커뮤니티가 형성되어 있어 학습 자료와 도움을 쉽게 구할 수 있습니다.

## 설치 방법

### Windows에서 설치하기

1. [python.org](https://www.python.org) 공식 사이트에 접속합니다.
2. Downloads 메뉴에서 최신 버전을 다운로드합니다.
3. 설치 파일을 실행할 때 반드시 **"Add Python to PATH"** 옵션을 체크합니다.
4. "Install Now"를 클릭하여 설치를 진행합니다.

설치가 완료되면 명령 프롬프트(CMD)를 열고 다음 명령어로 정상 설치 여부를 확인합니다.

```python
# 명령 프롬프트에서 실행
python --version
# 출력 예시: Python 3.12.0
```

### Mac에서 설치하기

Mac에는 기본적으로 Python이 설치되어 있을 수 있지만, 최신 버전을 사용하려면 별도로 설치하는 것이 좋습니다.

1. Homebrew가 설치되어 있다면 터미널에서 `brew install python3` 명령어를 실행합니다.
2. 또는 python.org에서 Mac용 설치 파일을 다운로드하여 설치합니다.

## VS Code 설정

파이썬 개발에 가장 널리 사용되는 에디터는 **Visual Studio Code(VS Code)** 입니다. 다음과 같이 설정하면 쾌적한 개발 환경을 구축할 수 있습니다.

1. [code.visualstudio.com](https://code.visualstudio.com)에서 VS Code를 다운로드하여 설치합니다.
2. 확장 프로그램(Extensions) 탭에서 **Python** 확장을 검색하여 설치합니다.
3. **Pylint** 또는 **Ruff** 를 설치하여 코드 품질을 관리합니다.
4. 설정에서 기본 포매터를 **Black** 으로 지정하면 코드 스타일을 자동으로 정리해줍니다.

## 변수와 자료형

파이썬에서 변수는 별도의 타입 선언 없이 값을 할당하면 자동으로 타입이 결정됩니다.

```python
# 변수 선언
name = "홍길동"       # 문자열(str)
age = 25              # 정수(int)
height = 175.5        # 실수(float)
is_student = True     # 불리언(bool)

# 변수 출력
print(f"이름: {name}")
print(f"나이: {age}세")
print(f"키: {height}cm")
print(f"학생 여부: {is_student}")
```

### 주요 자료형 정리

| 자료형 | 설명 | 예시 |
|--------|------|------|
| `int` | 정수 | `42`, `-10` |
| `float` | 실수 | `3.14`, `-0.5` |
| `str` | 문자열 | `"안녕하세요"` |
| `bool` | 참/거짓 | `True`, `False` |
| `list` | 리스트 | `[1, 2, 3]` |
| `dict` | 딕셔너리 | `{"key": "value"}` |

## 조건문과 반복문

### 조건문 (if문)

조건문은 특정 조건에 따라 다른 코드를 실행할 때 사용합니다.

```python
score = 85

if score >= 90:
    print("A학점입니다.")
elif score >= 80:
    print("B학점입니다.")
elif score >= 70:
    print("C학점입니다.")
else:
    print("D학점 이하입니다.")
```

### 반복문 (for문, while문)

반복문은 동일한 작업을 여러 번 수행할 때 사용합니다.

```python
# for문: 리스트 순회
fruits = ["사과", "바나나", "딸기", "포도"]
for fruit in fruits:
    print(f"나는 {fruit}을(를) 좋아합니다.")

# for문: range 활용
for i in range(1, 6):
    print(f"{i}번째 반복입니다.")

# while문
count = 0
while count < 5:
    print(f"현재 카운트: {count}")
    count += 1
```

## 함수 기초

함수는 특정 작업을 수행하는 코드 블록을 재사용 가능하게 묶어놓은 것입니다. `def` 키워드를 사용하여 함수를 정의합니다.

```python
# 기본 함수 정의
def greet(name):
    return f"안녕하세요, {name}님! 반갑습니다."

# 함수 호출
message = greet("홍길동")
print(message)

# 기본값이 있는 매개변수
def introduce(name, age, city="서울"):
    print(f"저는 {city}에 사는 {age}살 {name}입니다.")

introduce("김철수", 30)
introduce("이영희", 25, "부산")

# 여러 값을 반환하는 함수
def calculate(a, b):
    add = a + b
    subtract = a - b
    multiply = a * b
    return add, subtract, multiply

result = calculate(10, 3)
print(f"합: {result[0]}, 차: {result[1]}, 곱: {result[2]}")
```

## 첫 프로젝트: 간단한 계산기

지금까지 배운 내용을 활용하여 간단한 계산기 프로그램을 만들어 봅시다. 사용자로부터 두 개의 숫자와 연산자를 입력받아 결과를 출력하는 프로그램입니다.

```python
def calculator():
    print("=" * 30)
    print("  간단한 계산기 프로그램")
    print("=" * 30)

    while True:
        try:
            num1 = float(input("첫 번째 숫자를 입력하세요: "))
            operator = input("연산자를 입력하세요 (+, -, *, /): ")
            num2 = float(input("두 번째 숫자를 입력하세요: "))

            if operator == "+":
                result = num1 + num2
            elif operator == "-":
                result = num1 - num2
            elif operator == "*":
                result = num1 * num2
            elif operator == "/":
                if num2 == 0:
                    print("0으로 나눌 수 없습니다.")
                    continue
                result = num1 / num2
            else:
                print("올바른 연산자를 입력해주세요.")
                continue

            print(f"결과: {num1} {operator} {num2} = {result}")

        except ValueError:
            print("숫자를 올바르게 입력해주세요.")

        again = input("계속하시겠습니까? (y/n): ")
        if again.lower() != "y":
            print("프로그램을 종료합니다.")
            break

# 프로그램 실행
calculator()
```

이 프로젝트에서는 **함수 정의**, **반복문**, **조건문**, **예외 처리**, **사용자 입력** 등 여러 가지 기본 개념을 종합적으로 활용하고 있습니다.
