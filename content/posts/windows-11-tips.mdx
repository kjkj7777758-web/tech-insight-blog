## 들어가며

윈도우 11은 이전 버전에 비해 상당히 많은 기능이 추가되고 개선되었지만, 대부분의 사용자들은 기본적인 기능만 사용하고 있습니다. 이 글에서는 업무 효율을 크게 높여줄 수 있는 숨겨진 기능과 유용한 설정 팁 15가지를 소개합니다.

## 1. Snap Layouts 활용법

Snap Layouts는 윈도우 11에서 가장 유용하게 추가된 기능 중 하나입니다. 창의 최대화 버튼 위에 마우스를 올리면 다양한 레이아웃 옵션이 표시되며, 원하는 배치를 클릭하면 창이 자동으로 해당 위치에 배치됩니다.

### 활용 방법

- **최대화 버튼에 마우스 올리기**: 4~6가지 레이아웃 옵션이 표시됩니다.
- **키보드 단축키**: `Win + Z`를 누르면 Snap Layouts 메뉴를 키보드로 조작할 수 있습니다.
- **드래그 방식**: 창을 화면 상단으로 드래그하면 Snap Layouts 선택 화면이 나타납니다.
- **Snap Groups**: 함께 배치한 앱들이 그룹으로 기억되어 작업 표시줄에서 그룹 단위로 전환할 수 있습니다.

모니터가 넓을수록 더 많은 레이아웃 옵션이 제공되므로, 울트라와이드 모니터 사용자에게 특히 유용합니다.

## 2. 가상 데스크톱

가상 데스크톱은 하나의 모니터에서 여러 개의 독립적인 작업 공간을 만들어 사용할 수 있는 기능입니다.

### 핵심 단축키

- **새 데스크톱 만들기**: `Win + Ctrl + D`
- **데스크톱 간 전환**: `Win + Ctrl + 좌/우 화살표`
- **현재 데스크톱 닫기**: `Win + Ctrl + F4`
- **데스크톱 목록 보기**: `Win + Tab`

### 활용 팁

업무용, 개인용, 프로젝트별로 데스크톱을 분리하면 작업 집중도가 크게 향상됩니다. 각 데스크톱에 다른 배경화면을 설정하여 시각적으로 구분하는 것도 좋은 방법입니다. 데스크톱 이름도 변경할 수 있어 용도를 명확히 구분할 수 있습니다.

## 3. PowerToys 필수 기능

**PowerToys**는 마이크로소프트에서 공식 제공하는 고급 사용자를 위한 유틸리티 모음입니다. Microsoft Store에서 무료로 설치할 수 있으며, 다음과 같은 강력한 기능들을 제공합니다.

### FancyZones

Snap Layouts보다 더 자유로운 창 배치가 가능합니다. 사용자가 직접 영역을 설계하여 원하는 레이아웃을 만들 수 있으며, `Shift` 키를 누른 채 창을 드래그하면 사전에 정의한 영역에 맞춰 배치됩니다.

### PowerToys Run

`Alt + Space`로 실행하는 빠른 실행 도구입니다. 앱 실행, 파일 검색, 계산, 단위 변환, 시스템 명령 등 다양한 작업을 키보드만으로 빠르게 수행할 수 있습니다. macOS의 Spotlight와 유사한 기능을 제공합니다.

### Color Picker

`Win + Shift + C`로 화면 어디서든 색상 값을 추출할 수 있습니다. 디자인 작업이나 웹 개발 시 원하는 색상 코드를 빠르게 확인할 수 있어 매우 편리합니다.

## 4. 클립보드 히스토리

`Win + V`를 누르면 클립보드 히스토리 기능을 사용할 수 있습니다. 이전에 복사한 텍스트, 이미지, 파일 등의 이력을 확인하고 원하는 항목을 선택하여 붙여넣을 수 있습니다.

### 주요 기능

- **최대 25개 항목 저장**: 최근 복사한 항목을 25개까지 보관합니다.
- **고정 기능**: 자주 사용하는 항목을 고정하면 히스토리가 지워져도 유지됩니다.
- **클라우드 동기화**: 마이크로소프트 계정으로 로그인하면 여러 기기 간에 클립보드를 동기화할 수 있습니다.
- **이모지 및 특수문자**: 클립보드 패널에서 이모지, GIF, 특수 기호도 검색하여 입력할 수 있습니다.

처음 사용할 때는 설정에서 클립보드 히스토리를 활성화해야 합니다. **설정 > 시스템 > 클립보드**에서 켤 수 있습니다.

## 5. Focus Sessions (집중 세션)

Focus Sessions는 일정 시간 동안 알림을 차단하고 집중할 수 있도록 도와주는 기능입니다.

### 설정 방법

1. 시계 앱을 열거나 알림 센터에서 포커스 버튼을 클릭합니다.
2. 집중 시간을 설정합니다 (25분, 50분 등 포모도로 기법 적용 가능).
3. 세션을 시작하면 알림이 자동으로 차단됩니다.

### 연동 기능

- **Microsoft To Do**: 집중 세션 중 수행할 작업을 선택할 수 있습니다.
- **Spotify**: 집중에 도움이 되는 음악을 자동으로 재생할 수 있습니다.
- **일일 목표**: 하루에 집중한 시간의 목표를 설정하고 달성률을 추적할 수 있습니다.

## 6. 터미널 활용

윈도우 11에는 **Windows Terminal**이 기본 내장되어 있습니다. 이전의 명령 프롬프트나 PowerShell보다 훨씬 강력하고 편리한 기능을 제공합니다.

### 주요 기능

- **탭 지원**: 여러 셸을 탭으로 관리하여 하나의 창에서 전환하며 사용할 수 있습니다.
- **분할 창**: `Alt + Shift + D`로 현재 창을 분할하여 두 개의 셸을 동시에 볼 수 있습니다.
- **다양한 프로필**: PowerShell, 명령 프롬프트, WSL(리눅스), Azure Cloud Shell 등을 하나의 앱에서 사용합니다.
- **커스터마이징**: 글꼴, 색상, 배경 투명도, 아크릴 효과 등을 자유롭게 설정할 수 있습니다.

### 관리자 권한 실행

작업 표시줄의 터미널 아이콘을 `Ctrl + Shift + 클릭`하면 관리자 권한으로 바로 실행할 수 있습니다.

## 7. 파일 탐색기 숨겨진 기능

### 탭 기능

파일 탐색기에서도 브라우저처럼 탭을 사용할 수 있습니다. `Ctrl + T`로 새 탭을 열고, `Ctrl + W`로 탭을 닫을 수 있습니다. 여러 폴더를 동시에 열어두고 전환하며 작업할 수 있어 파일 관리가 훨씬 편리합니다.

### 빠른 접근 팁

- **주소 표시줄에 경로 직접 입력**: 원하는 폴더로 빠르게 이동할 수 있습니다.
- **미리보기 패널**: `Alt + P`를 누르면 파일 내용을 미리볼 수 있는 패널이 표시됩니다.
- **세부 정보 패널**: `Alt + Shift + P`로 파일의 상세 정보를 확인할 수 있습니다.
- **압축 파일 지원**: 별도 프로그램 없이 ZIP, TAR, 7z, RAR 등의 압축 파일을 열고 해제할 수 있습니다.

## 8. 키보드 단축키 모음

윈도우 11에서 반드시 알아야 할 핵심 단축키를 정리하였습니다.

### 시스템 단축키

- **Win + A**: 빠른 설정 패널 열기
- **Win + N**: 알림 센터 열기
- **Win + W**: 위젯 패널 열기
- **Win + H**: 음성 입력 기능 활성화
- **Win + Shift + S**: 화면 캡처 도구 실행
- **Win + .** (마침표): 이모지 및 특수문자 입력 패널

### 창 관리 단축키

- **Win + 화살표**: 창을 화면의 왼쪽, 오른쪽, 최대화, 최소화
- **Alt + Tab**: 실행 중인 앱 간 전환
- **Win + Tab**: 작업 보기 (가상 데스크톱 포함)
- **Win + D**: 바탕화면 보기 토글
- **Win + L**: 화면 잠금

### 생산성 단축키

- **Win + E**: 파일 탐색기 열기
- **Win + I**: 설정 열기
- **Win + P**: 디스플레이 전환 (복제, 확장 등)
- **Ctrl + Shift + Esc**: 작업 관리자 열기

## 9. 음성 입력 기능

`Win + H`를 누르면 음성 입력 기능이 활성화됩니다. 한국어를 포함한 다양한 언어를 지원하며, 마이크에 대고 말하면 텍스트로 자동 변환됩니다. 긴 문서를 작성하거나 타이핑이 어려운 상황에서 유용합니다.

### 활용 팁

- 구두점도 음성으로 입력할 수 있습니다 (예: "마침표", "쉼표", "물음표").
- 메모장, 워드, 이메일 등 텍스트 입력이 가능한 모든 곳에서 사용할 수 있습니다.
- 인터넷 연결이 필요하며 조용한 환경에서 인식률이 높습니다.

## 10. 라이브 캡션

라이브 캡션은 컴퓨터에서 재생되는 모든 오디오에 대해 실시간 자막을 생성하는 기능입니다. 영어 콘텐츠를 시청할 때 청각 보조 도구로 활용하거나, 공공장소에서 소리를 켜지 못할 때 유용합니다. **설정 > 접근성 > 캡션**에서 활성화할 수 있습니다.

## 11. 동적 새로 고침 빈도

노트북 사용자라면 **동적 새로 고침 빈도(DRR)** 기능을 활용할 수 있습니다. 이 기능은 화면에 움직임이 있을 때만 높은 주사율(예: 120Hz)을 사용하고, 정적인 화면에서는 주사율을 낮춰 배터리 소모를 줄입니다. **설정 > 시스템 > 디스플레이 > 고급 디스플레이**에서 "동적"을 선택하면 됩니다.

## 12. 스마트 앱 컨트롤

스마트 앱 컨트롤은 신뢰할 수 없는 앱이나 잠재적으로 위험한 소프트웨어의 실행을 차단하는 보안 기능입니다. 마이크로소프트의 클라우드 인텔리전스를 기반으로 앱의 안전성을 판단하며, 악성 소프트웨어로부터 시스템을 보호합니다.

## 13. 에너지 권장 사항

**설정 > 시스템 > 전원 및 배터리 > 에너지 권장 사항**에서 시스템의 에너지 효율을 높이는 최적화 설정을 확인할 수 있습니다. 화면 밝기, 절전 모드 시간, 디스플레이 끄기 시간 등 에너지 절약을 위한 설정이 권장 사항으로 제시되며, 원클릭으로 적용할 수 있습니다.

## 14. 개인정보 설정 최적화

윈도우 11은 기본적으로 다양한 데이터를 수집합니다. 개인정보 보호를 강화하려면 다음 설정을 확인하는 것이 좋습니다.

### 확인해야 할 설정 항목

- **설정 > 개인 정보 및 보안 > 일반**: 광고 ID, 언어 목록 접근, 앱 실행 추적 등의 옵션을 비활성화합니다.
- **설정 > 개인 정보 및 보안 > 진단 및 피드백**: 선택적 진단 데이터 전송을 비활성화합니다.
- **설정 > 개인 정보 및 보안 > 활동 기록**: 활동 기록 저장 옵션을 비활성화합니다.
- **설정 > 개인 정보 및 보안 > 검색 권한**: 클라우드 콘텐츠 검색과 검색 기록 옵션을 조정합니다.

## 15. 작업 표시줄 사용자 정의

작업 표시줄을 자신의 작업 스타일에 맞게 최적화할 수 있습니다.

### 주요 설정

- **작업 표시줄 정렬**: 기본 가운데 정렬을 왼쪽 정렬로 변경할 수 있습니다. **설정 > 개인 설정 > 작업 표시줄 > 작업 표시줄 동작**에서 변경합니다.
- **불필요한 아이콘 제거**: 위젯, 검색, 작업 보기 등 사용하지 않는 아이콘을 숨길 수 있습니다.
- **시스템 트레이 정리**: 알림 영역에 표시할 아이콘을 선택하여 깔끔하게 유지합니다.
- **작업 표시줄 자동 숨기기**: 화면 공간을 최대한 활용하고 싶다면 자동 숨기기를 활성화합니다.

## 결론

윈도우 11에는 업무 효율과 사용 편의성을 높여주는 기능이 생각보다 많이 숨어 있습니다. 이 글에서 소개한 15가지 기능을 모두 한꺼번에 적용할 필요는 없습니다. 자신의 업무 스타일과 필요에 맞는 기능부터 하나씩 적용해 보면서 점차 활용 범위를 넓혀 나가는 것을 권장합니다.

특히 **Snap Layouts**, **가상 데스크톱**, **클립보드 히스토리**, **키보드 단축키**는 한 번 익히면 일상적으로 사용하게 되는 핵심 기능이므로 우선적으로 학습하는 것이 좋습니다. PowerToys까지 활용하면 윈도우 11의 생산성을 극대화할 수 있습니다.

운영체제의 기능을 충분히 활용하는 것만으로도 별도의 유료 도구 없이 작업 환경을 크게 개선할 수 있습니다. 새로운 윈도우 업데이트마다 유용한 기능이 추가되고 있으므로 정기적으로 업데이트하고 새로운 기능을 확인해 보시기 바랍니다.
